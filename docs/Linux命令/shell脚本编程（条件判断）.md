##### 运行方法
```angular2html
    $ ./test  #test 可执行脚本程序
    $ sh test #test shell脚本
```
备注：如果脚本运行前必须键入路径名，或者shell结果通知无法找到命令，则需要在.profile PATH下加入用户可执行程序目录。
` PATH=$PATH:$HOME/bin `

###### 条件测试
`[ -d 1.txt ] #判断条件写法`
```angular2html
    #（字符串判断）
    #判断两个变量是否相等
    $ PARM_1="1";
    $ PARM_2="2";
    $ [ "$PARM_1" = "$PARM_2"]
    
    #数值判断
    -eq 数值相等
    -ne 数值不相等
    -gt 大于
    -lt 小于
    -le 小于等于
    -ge 大于等于
    #expr  操作数值表达式
    expr 10 + 10 
    #expr 作条件表达式处理
    PARM_1=2
    expr $PARM_1 = "2" #成功返回 1 失败返回0
```

|符号|注释|
|---|---|
|-d|是否为目录|
|-f|是否为文件|
|-L|是否为符号链接|
|-r|是否可读|
|-s|是否长度大于0、非空|
|-w|是否可写|
|-u|文件有suid位置|
|-x|可执行|
|-t|是否为交互式（终端模式）非交互模式(cron、at) |
|-z|判断变量是否被设置|
